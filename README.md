# PostgreSQL Coursework - Database Management Systems

This repository contains all my coursework assignments for the **Database Management Systems** course using **PostgreSQL**.  
Each folder corresponds to a set of checkpoints covering key topics in relational database design and SQL querying.

---

## Project Structure

The repository is organized as follows:

    /Checkpoint-1-2
    /Checkpoint-3-4
    /Checkpoint-5-6
    ...

Each folder contains SQL scripts and related files for the corresponding checkpoints.
